# Edunity Parent Request App

Standalone Next.js app for parent request intake only.

## Local run

1. Copy `.env.example` to `.env.local`
2. Fill in Firebase `NEXT_PUBLIC_FIREBASE_*` values
3. Install and run:
   - `npm install`
   - `npm run dev`

## Firestore target

Submits to collection: `parent_requests`

## Vercel deploy

Set Vercel project root directory to:

`parent-request-app`
